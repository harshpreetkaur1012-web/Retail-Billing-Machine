📌 About Project

A desktop-based billing system built using Python and Tkinter. It automates billing for retail shops with features like bill generation, tax calculation, saving invoices, printing, and email sending.

🚀 Features

🛒 Category-wise Billing
🧮 Auto Price Calculation
📧 Email Invoice
🖨️ Print Bills
💾 Save & Search Bills
🧹 Reset System

🛠️ Technologies Used

Python Tkinter SMTP OS Module GitHub

2. Navigate to folder
3. Run: python main.py
